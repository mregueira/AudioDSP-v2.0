# AudioDSP-v2.0
## Description
Audio DSP - 2.0 - Second version attempt of final product

## Features
- DC Input +6V/1A
- Audio Channels: up to 5.1
- SigmaDSP ADAU1701
   - I2C control for DSP
- Sampling frequency (default): 48KHz
- 24 bit ADC PCM1862 x3
- 24 bit DAC PCM1681 x1
- ST uC STM32F401RxT6 with Serial Wire programming
- LED indicators
