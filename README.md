# AudioDSP-v1.0
## Description
Audio DSP - 1.0 - First version attempt of final product

## Features
- DC Input +5V/1A
- Audio Channels: up to 5.1
- SigmaDSP ADAU1701
   - I2C control for DSP
   - 24 bit ADC inputs x2
   - 24 bit DAC outputs x4
- Sampling frequency (default): 48KHz
- ST uC STM32F401RxT6 with Serial Wire programming
- LED indicators
